# RISC-V-Single-Cycle-in-Verilog

Verilog implementation of complete single-cycle RISC-V processor supporting R/I/S-type instructions. Full datapath with PC, instruction/data memory, 32-entry register file, ALU, and control unit. Successfully simulated in Xilinx Vivado.
